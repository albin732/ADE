# 🤝 Contributing to ADE

---

## 🎯 Goal

Maintain a **controlled AI development system**.

---

## 📌 Rules

* Keep changes minimal
* Do NOT rewrite entire project
* Avoid breaking working code

---

## 📂 Scope

* Modify only relevant files
* Avoid global side-effects

---

## 🧪 Testing

```bash
pytest -v
```

---

## 🚫 Do NOT commit

* `.venv`
* `projects/`
* `env.sh`

---

## 🚀 Workflow

1. Fork repo
2. Create branch
3. Make changes
4. Submit PR

---

## 💡 Focus Areas

* precision mode
* loop improvements
* multi-framework support

---

## 🧠 Philosophy

Controlled, safe, predictable AI-assisted development.
